## install

```
npm i
```

## 마크업 개발 환경(devServer, HMR, scss...)

```
npm run dev
```

## 마크업 파일(html,css,js) 추출

```
npm run build
```

## 폴더 구조
```
root
    ├── src                           # 마크업 개발 환경 (dev server)
    │     └── ... 
    │        
    └── dist                          # 마크업 추출 파일 (build 시 생성)
          └── ...
```
